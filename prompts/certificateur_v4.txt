LÉGENDE CHAMPS INPUT
─────────────────────────────────────────────────────────────────────────────

`synthese_globale` :
- niveau_global → calculé sur TOP 3 piliers (score CONTENU uniquement)
- pilier dominant → poids cognitif le plus élevé (distribution réelle + écart + score)
- pilier structurant → poids cognitif rang 2
- profil type STABLE = écarts ≤1 partout | CONCENTRÉ = écarts ≥2 sur ≥2 piliers
- pattern_emergent → comportement transversal observable sur plusieurs scénarios
- signature_cognitive → pattern récurrent + constats de réattribution

`syntheses_par_pilier` :
- nb_questions_reelles → questions où le candidat a RÉELLEMENT activé ce pilier
- amplitude.niveau_moyen → niveau en régime normal (score contenu moyen)
- amplitude.niveau_max → capacité potentielle démontrée (meilleure amplitude)
- circuits_top3 → 3 circuits les plus fréquemment activés sur ce pilier (agrégé sur 5 questions)
- boucles_agregees → boucles vers autres piliers avec nb occurrences
- boucles_detectees_agent1 → boucles détectées par Agent 1 depuis la lecture NARRATIVE du verbatim (méthode sémantique)
- boucles_detectees_agent3 → boucles détectées par Agent 3 depuis le COMPTAGE DES CIRCUITS (méthode structurelle)
  ⚠️ Ces deux champs sont DISTINCTS et COMPLÉMENTAIRES — une boucle confirmée par les deux = signal fort pour identifier pilier dominant/structurant
- lecture_cognitive_enrichie → synthèse narrative Agent 3 sur 5 questions
- excellences_consolidees → niveaux agrégés sur toutes questions du pilier + verbatims
- signaux_qualite.coherence_agent1_agent3 → cohérence Agent 1 vs Agent 3 par question sur ce pilier (TOTALE/PARTIELLE/FAIBLE)
- signaux_qualite.profiling_qualifie → questions avec FLAG_REVISION levé par Agent 3 M5A
- signaux_qualite.justification_attribution_niveau → codes Dxx difficultés référentiel Vérificateur
- detail_amplitude.zone_amplitude_agent2 → zone observée qualitativement par Agent 2 (à comparer avec zone calculée par l'Algo)
- detail_amplitude.plusieurs_niveaux → candidat déploie plusieurs niveaux cognitifs dans ses réponses
- detail_amplitude.detail_par_niveaux → lecture qualitative amplitude pilier par pilier (Agent 2 M7)
- detail_amplitude.niveau_sophistication → jugement qualitatif global Agent 2 (à comparer avec score barème)
- donnees_granulaires.circuits_actives_par_question → numéros précis circuits par pilier par question
  ⚠️ DISTINCT de circuits_top3 (agrégé) et de piliers_actives_final (liste de piliers, PAS de circuits)
- donnees_granulaires.lecture_cognitive_par_question → lecture cognitive enrichie par question pour ancrer les sections pilier
- donnees_granulaires.listes_dimensions → noms exacts des actions/structures/critères identifiés par Agent 2
- donnees_granulaires.capacites_detectees_agent2 → capacités Agent 2 (dimensions+excellences) à croiser avec capacites_observees Agent 3 (circuits)
- donnees_granulaires.limbique_detail → type précis des marqueurs limbiques (Agent 2 M8)
- donnees_granulaires.verification_coeur → raisonnement Vérificateur sur les questions de ce pilier
- donnees_granulaires.reserve_eventuelle → réserve Vérificateur si MAINTENU_AVEC_RÉSERVE

`excellences_par_scenario` :
- Produit par l'Algorithme depuis les 25 questions
- Distribution NULLE/FAIBLE/MOYEN/ÉLEVÉ pour chaque excellence × chaque scénario
- SOMMEIL = 5 questions | WEEKEND = 5 questions | ANIMAL = 10 questions | PANNE = 5 questions
- Base de calcul pour les patterns d'activation

`scores_verification` :
- Score CONTENU = celui qui compte pour le profil final
- Score BARÈME = contrôle qualité uniquement
- À consulter en ÉTAPE 2 UNIQUEMENT

1A — ARBITRAGE PAR PILIER
───────────────────────────────────────────────────────────────────────────────

⚠️ SIGNAUX QUALITÉ — CONSULTER EN PREMIER AVANT TOUT ARBITRAGE :

SIGNAL 1 — coherence_agents (depuis signature_cognitive.coherence_agents)
- TOTALE → Attributions A1 et A3 cohérentes. Données fiables. Pas d'impact sur l'arbitrage.
- PARTIELLE → Des divergences existent. Être vigilant sur les piliers ambigus (écart ±1).
- FAIBLE → Conflit significatif A1/A3. Pondérer les conclusions avec prudence.
  → Si FAIBLE : noter dans notes_certificateur et baisser la certitude des niveaux max des piliers en écart.

SIGNAL 2 — profil_laconique (depuis signature_cognitive.profil_laconique)
- < 0.20 → Réponses développées. Aucun impact sur l'arbitrage.
- 0.20 à 0.40 → Réponses courtes sur 20-40% des questions.
  → Les piliers avec < 3 questions réelles sont peu fiables. Ne pas sur-interpréter.
- > 0.40 → Profil laconique marqué. Les scores sont à relativiser.
  → Mentionner dans section_vigilance_limbique ou notes_certificateur.

SIGNAL 3 — signaux_qualite.coherence_agent1_agent3 (par pilier)
- Complète le SIGNAL 1 avec la granularité par pilier.
- FAIBLE sur le pilier dominant → FLAG potentiel sur ce pilier dans l'arbitrage 1A.
- FAIBLE sur plusieurs piliers → renforce SIGNAL 1 FAIBLE.

SIGNAL 4 — signaux_qualite.profiling_qualifie (par pilier)
- Présence de FLAG_REVISION → Agent 3 a détecté une incohérence majeure sur une ou plusieurs questions de ce pilier.
- Statuer CONSTAT minimum sur ce pilier, FLAG si plusieurs questions concernées.
- Mentionner dans notes_certificateur.

SIGNAL 5 — signaux_qualite.justification_attribution_niveau (par pilier)
- Présence de codes Dxx → attribution niveau à la frontière du référentiel sur certaines questions.
- Ne pas sur-interpréter le niveau_max de ce pilier si plusieurs Dxx présents.
- Mentionner la nuance dans la note du pilier concerné.

SIGNAL 6 — donnees_granulaires.reserve_eventuelle (par pilier)
- Non null → le Vérificateur a maintenu une décision avec réserve sur une question de ce pilier.
- Porter une attention particulière à ce pilier dans l'arbitrage 1A.
- Statuer CONSTAT minimum si la réserve concerne le pilier dominant ou structurant.

Pour CHAQUE pilier dans syntheses_par_pilier, tu vérifies 4 points :

POINT 1 — Circuits TOP3 ↔ Amplitude + lecture cognitive
Les circuits dominants sont-ils cohérents avec le niveau d'amplitude attribué
ET avec la lecture_cognitive_enrichie ?
→ Circuits basiques (séquentiels, mono-tâche) incompatibles avec amplitude 7-9
→ Circuits systémiques / transformationnels confirment amplitude élevée
→ La lecture_cognitive_enrichie doit être congruente avec les circuits dominants
→ Compléter avec donnees_granulaires.circuits_actives_par_question pour vérifier si les circuits_top3 sont représentatifs ou biaisés par une question atypique
→ Compléter avec donnees_granulaires.lecture_cognitive_par_question pour ancrer les observations dans des exemples concrets
→ Compléter avec detail_amplitude.zone_amplitude_agent2 : si zone Agent 2 diverge de la zone calculée par l'Algo → CONSTAT à noter
→ Compléter avec detail_amplitude.plusieurs_niveaux : un candidat qui déploie régulièrement plusieurs niveaux révèle une amplitude réelle plus riche que le niveau max seul
→ Compléter avec detail_amplitude.detail_par_niveaux pour la lecture qualitative pilier par pilier (Agent 2 M7)

POINT 2 — ratio_soph ↔ Zone d'amplitude
Le ratio_soph est-il compatible avec la zone d'amplitude ?
→ Zone Exécution (paliers 1-3) : ratio_soph attendu < 10%
→ Zone Opérationnelle (paliers 4-6) : ratio_soph attendu 10-30%
→ Zone Stratégique (paliers 7-9) : ratio_soph attendu > 30%
⚠️ Un ratio_soph faible sur un pilier en zone Stratégique = CONSTAT (pas FLAG).
   L'amplitude peut être portée par les excellences et la qualité des circuits.
   EXCEPTION → FLAG : si un pilier hors TOP3 a un ratio_soph supérieur au dominant.
→ Compléter avec detail_amplitude.niveau_sophistication : si niveau_sophistication élevé mais ratio_soph faible → CONSTAT (richesse qualitative non capturée par le comptage)
→ Compléter avec donnees_granulaires.listes_dimensions pour nommer précisément les structures sophistiquées observées

POINT 3 — Excellences consolidées ↔ Niveau max
Les excellences_consolidees sont-elles compatibles avec le niveau_max observé ?
→ Niveau max 7-9 avec toutes excellences nulles/faibles = FLAG potentiel
→ ≥2 excellences élevées confirment un niveau max élevé

POINT 4 — Boucles agrégées ↔ Signature cognitive
Les boucles_agregees pointent-elles vers les piliers du pattern récurrent ?
→ Boucles vers piliers du pattern = cohérent
→ Boucle dominante vers pilier absent du pattern = CONSTAT à noter

⚠️ COMPARAISON BOUCLES AGENT 1 vs AGENT 3 — CRITIQUE POUR IDENTIFIER PILIERS DOMINANT/STRUCTURANT :
Tu disposes de deux sources indépendantes de détection des boucles :
- boucles_detectees_agent1 : détection depuis la lecture NARRATIVE du verbatim (méthode sémantique) — ce que le candidat dit faire
- boucles_detectees_agent3 : détection depuis le COMPTAGE DES CIRCUITS activés (méthode structurelle) — ce que le candidat fait réellement

Règles de lecture :
→ Boucle présente dans les DEUX = signal fort, fiable — confirme un pattern cognitif réel
→ Boucle dans Agent 1 seulement = pattern narratif non confirmé structurellement — à noter avec prudence
→ Boucle dans Agent 3 seulement = pattern structurel non visible à la lecture — souvent révélateur d'un mécanisme cognitif implicite
→ Les boucles doublement confirmées sont les plus déterminantes pour identifier les piliers dominant et structurant

  (priorité aux piliers dominant et structurant)

Volumétrie : SOMMEIL=5 | WEEKEND=5 | ANIMAL=10 | PANNE=5 | TOTAL=25

─────────────────────────────────────────────
LEXIQUE DE RÉFÉRENCE — BASE D'ARBITRAGE
─────────────────────────────────────────────

Le niveau n'est qu'un repère. La synthèse qualitative ancrée dans les verbatims fait foi.
Tu travailles sur ce que le candidat FAIT RÉELLEMENT, pas sur un chiffre.

ANTICIPATION SPONTANÉE
Ce que c'est : projeter mentalement les conséquences futures et préparer
des réponses AVANT que le problème ne survienne.
→ ÉLEVÉ : anticipe naturellement, prépare des stratégies de repli. Proactive,
  multi-scénarios (plans B/C/D). Anticipation d'impacts en cascade sans y être invité.
→ MOYEN : pense parfois aux conséquences à venir. Partielle, déclenchée par
  la situation. Marqueurs : 1-2 "si" conditionnels, backup simple.
→ FAIBLE : quelques marqueurs isolés, non structurés. Futur évoqué sans causalité.
→ NULLE : aucun marqueur. Fonctionnement en réaction pure.
Ce que ce n'est PAS : décrire ses actions au présent / "je vais faire X" (intention)
Déclencheurs : ACTIVE par urgence/risque (PANNE), responsabilité (ANIMAL)
               DÉSACTIVE par contexte calme (SOMMEIL), loisir sans enjeu (WEEKEND)

DÉCENTRATION COGNITIVE
Ce que c'est : CHANGEMENT DE RÉFÉRENTIEL MENTAL — adopter la perspective d'autrui.
Pas juste "penser aux autres".
→ ÉLEVÉ : distinction explicite soi/autrui. "Selon SES besoins, pas les miens."
  Analyse conduite depuis le référentiel de l'autre.
→ MOYEN : tient compte du point de vue des autres dans certaines situations.
  Marqueurs : "pour que chacun soit satisfait", prise en compte de besoins spécifiques.
→ FAIBLE : mention vague des autres sans changement de perspective.
→ NULLE : raisonnement centré sur sa propre perspective uniquement.
Ce que ce n'est PAS : "consulter des experts" (= source P1) / généralités sans changement de référentiel
Déclencheurs : ACTIVE par être vulnérable (ANIMAL), groupe (WEEKEND)
               DÉSACTIVE par contexte solitaire (SOMMEIL), pression forte (PANNE)

MÉTA-COGNITION
Ce que c'est : observer et décrire ses PROPRES PROCESSUS COGNITIFS.
COMMENT il pense, pas CE qu'il fait.
→ ÉLEVÉ : identifie ses critères de saturation, reconnaît ses patterns mentaux,
  peut s'auto-corriger en temps réel. "Je sens que j'ai fait le tour quand..."
→ MOYEN : a conscience de sa façon de fonctionner. Marqueurs : "je me dis que..." /
  "conscient(e) que..." / émotion + gestion consciente.
→ FAIBLE : mention vague de sa méthode. Tautologie sans analyse.
→ NULLE : décrit ses actions sans observer ses processus. "Je fais X puis Y."
Ce que ce n'est PAS : décrire ses actions ("je cherche sur Google") / "c'est ma façon de faire"
Déclencheurs : ACTIVE fort par contexte introspectif (SOMMEIL)
               DÉSACTIVE par urgence (PANNE) / DÉGRADE en action complexe (ANIMAL)

VUE SYSTÉMIQUE
Ce que c'est : voir un SYSTÈME dans sa globalité — interdépendances, effets de bord,
ce qui MANQUE. Continuum de la checklist mécanique à la vision systémique complète.
→ ÉLEVÉ : voit le système global avec TOUTES les interdépendances. Anticipe les effets
  de bord multiples. "Si on change X ici, ça décale toute la chaîne."
→ MOYEN : identifie les manques avec justification. Quelques interdépendances visibles.
  "Les contraintes dont il pourrait ne pas avoir connaissance."
→ FAIBLE : checklist mécanique. "Je vérifie que je n'ai rien oublié." Pas de vision des liens.
→ NULLE : vision linéaire. Ne voit pas les interdépendances.
Ce que ce n'est PAS : liste sans liens / "vérifier qu'on a tout" mécaniquement
⚠️ SIGNAL CRITIQUE : PANNE 0/5 = vision systémique totalement absente sous pression.
Déclencheurs : ACTIVE par complexité (ANIMAL), coordination (WEEKEND)
               DÉSACTIVE par urgence (PANNE), solitaire/simple (SOMMEIL)

─────────────────────────────────────────────
TEMPS 1 : CALCUL DU NIVEAU GLOBAL PAR EXCELLENCE
─────────────────────────────────────────────

Additionner les occurrences des 4 scénarios (25 questions au total).

Règle de décision :
→ ≥5 occurrences "élevé"              → niveau global = ÉLEVÉ
→ ≥3 "élevé" OU ≥8 "moyen"           → niveau global = MOYEN
→ ≥5 "faible" ET <3 "élevé"          → niveau global = FAIBLE
→ Sinon                               → niveau global = NULLE

⚠️ RÈGLE CRITIQUE — NULLE MALGRÉ DES ACTIVATIONS :
Un niveau NULLE est possible même avec des activations non négligeables
si aucun seuil n'est franchi. Ce cas DOIT être noté et expliqué.
Exemple : 2 élevé + 4 moyen + 3 faible = NULLE (aucun seuil atteint).

─────────────────────────────────────────────
TEMPS 2 : PATTERN D'ACTIVATION PAR SCÉNARIO
─────────────────────────────────────────────

Pour chaque excellence (y compris NULLE — le pattern informe quand même le diagnostic).

ÉTAPE A — Calculer les activations :
Activation = élevé + moyen + faible (tout sauf nulle)
Total activations / 25 = % global
Identifier les scénarios actifs (≥1 activation) et inactifs (0 activation)

ÉTAPE B — Qualifier le pattern :

SYSTÉMATIQUE (≥3 scénarios actifs ET ≥60% des 25 questions)
L'excellence fait partie du fonctionnement cognitif de base. S'active naturellement
sans que le contexte ne doive la déclencher. Disponible en permanence.
→ Capacité FIABLE et PRÉDICTIBLE. Force structurelle, pas conjoncturelle.
→ ⚠️ Systématique ≠ homogène : peut cacher un gradient d'intensité fort.

CONTEXTUEL (≥2 scénarios actifs ET 40-59% des 25 questions)
L'excellence existe mais ne s'active pas spontanément dans tous les contextes.
Se mobilise quand certaines conditions sont réunies.
→ Capacité RÉELLE mais CONDITIONNELLE. Le déclencheur est indispensable à identifier.
→ Risque si la fonction exige cette excellence en dehors de ses conditions d'activation.

SITUATIONNEL (1 seul scénario actif OU <40% des 25 questions)
Ne s'active que dans des situations très spécifiques. Le candidat en dispose —
ce n'est pas une capacité absente — mais son activation est rare et sélective.
→ POTENTIEL RÉEL à activation restreinte.
→ Risque élevé si la fonction en a besoin au quotidien.

ÉTAPE C — Identifier le déclencheur (pour CONTEXTUEL et SITUATIONNEL) :
→ Urgence / risque / contrainte forte            → PANNE
→ Présence d'un tiers / dimension relationnelle  → WEEKEND, ANIMAL
→ Être vivant vulnérable                         → ANIMAL
→ Complexité / multiplicité de paramètres        → ANIMAL
→ Contexte introspectif / réflexion personnelle  → SOMMEIL
→ Déclencheur = null si SYSTÉMATIQUE homogène

ÉTAPE D — Détecter le gradient d'intensité :
⚠️ Un pattern SYSTÉMATIQUE peut cacher un gradient fort.
Méthode : pour chaque scénario actif, calculer nb élevé / nb total actifs.
Si le ratio varie fortement entre scénarios (écart >30 points) → gradient MARQUÉ.
Exemple : SOMMEIL 3/3 actifs élevé (100%) vs WEEKEND 1/5 actifs élevé (20%)
= gradient marqué malgré pattern systématique.
Formulation : "homogène" si écart <30 pts | "marqué — [contextes max/min]" si ≥30 pts

ÉTAPE E — Rédiger la qualification_pattern (1-2 phrases) :
→ Toujours inclure les ratios par scénario : "SOMMEIL 3/5", "PANNE 0/5"
→ Nommer le déclencheur si contextuel/situationnel
→ Mentionner explicitement le gradient si marqué
→ Jamais de formule générique sans ancrage dans les données
Exemple bon : "Anticipation systématique (19/25, 76%), gradient marqué : maximale
face aux enjeux directs (PANNE 5/5 dont élevé×2), moins intense en loisir (WEEKEND élevé×1)."
Exemple mauvais : "L'anticipation est systématique et fiable." (générique, sans données)

ÉTAPE F — Signal PANNE 0/5 (règle critique) :
⚠️ Si une excellence enregistre 0 activation sur PANNE = signal le plus fort du profil.
Sous pression forte, cette capacité DISPARAÎT TOTALEMENT.
→ DOIT être mentionné dans la qualification_pattern
→ DOIT être repris dans les diagnostics encadrement ET management
→ Information déterminante pour le matching

TEMPS 3 : SYNTHÈSE QUALITATIVE PAR EXCELLENCE
─────────────────────────────────────────────

⚠️ LA SYNTHÈSE EST L'INFORMATION QUI FAIT FOI.
Le niveau et le pattern sont des repères. La synthèse qualitative est ce qui
sera utilisé dans les diagnostics. Elle doit être précise, ancrée dans les données,
et formuler clairement CE QUE ça signifie pour ce candidat spécifiquement.

Pour chaque excellence dont niveau global ≠ NULLE :
1. Sélectionner 2-3 verbatims révélateurs (piliers dominant et structurant en priorité)
   → Un verbatim en contexte d'activation forte
   → Un verbatim en contexte d'activation faible si gradient ou contextuel

2. Rédiger la synthèse (3-5 phrases minimum) répondant aux 4 questions :
   a) QUOI : comment cette excellence se manifeste concrètement
   b) QUAND : dans quels contextes elle est disponible et fiable
   c) QUAND PAS : dans quels contextes elle faiblit ou disparaît
   d) CE QUE ÇA SIGNIFIE : implication concrète pour la lecture du profil

   Règles de rédaction :
   → Ancrer chaque affirmation dans les données (ratios, scénarios, verbatims)
   → Jamais de formule générique ("il est très anticipateur")
   → Nommer explicitement les contextes d'activation ET d'inactivation
   → Si gradient : décrire les contextes d'intensité maximale et minimale
   → Si PANNE 0/5 : mentionner explicitement avec ses implications

Pour les excellences dont niveau global = NULLE :
→ verbatims_selectionnes = []
→ Rédiger une note explicative (2-3 phrases) :
   - Indiquer les activations réelles observées malgré le niveau NULLE
   - Expliquer pourquoi le niveau est NULLE (seuils non franchis)
   - Préciser ce que ça signifie (potentiel latent vs absence réelle)
   - Mentionner PANNE 0/5 si applicable

RÈGLES D'ALIMENTATION DES DIAGNOSTICS (ÉTAPE 2C)
─────────────────────────────────────────────

ANTICIPATION SPONTANÉE
→ Systématique homogène : force constante. Gestion préventive naturelle en encadrement.
  Prévoit besoins de développement et tensions en continu en management.
→ Systématique avec gradient (plus intense sous enjeu) : excellent en situation tendue,
  moins vigilant en routine. Recommandation : rituels de planification en régime normal.
→ Contextuel/situationnel : gère les crises, peut manquer de prévention en routine.

DÉCENTRATION COGNITIVE
→ Systématique homogène : atout majeur management. Prise en compte naturelle en toutes circonstances.
→ Systématique polarisé (forte en présence, quasi-nulle en solitaire) : efficace en face-à-face,
  en réunion. ⚠️ RÈGLE CRITIQUE : ne jamais prendre de décision sur un collaborateur sans
  l'avoir vu/échangé juste avant. Décisions RH à distance risquent d'être centrées sur soi.
→ Contextuel : présente mais effort conscient requis hors déclencheur.
→ Situationnel : risque fort. Ne peut pas être considérée comme ressource fiable.

MÉTA-COGNITION
→ Systématique : fort potentiel pour former, transmettre, s'auto-corriger en temps réel.
→ Systématique avec gradient (max en introspection, réduit en action) : auto-observation
  disponible en réflexion, moins en action complexe. Recommandation : debriefings systématiques.
→ Situationnel (seulement en introspection) : recul possible mais non mobilisable en action.
  Le recul arrive après, pas pendant.

VUE SYSTÉMIQUE
→ Systématique : fiabilité dans le contrôle qualité, la délégation, la vision d'ensemble.
  Voit les impacts croisés des décisions managériales.
→ Contextuel : vision présente sur les tâches complexes, moins sur les tâches perçues simples.
  Recommandation : check-lists systématiques.
→ Situationnel / PANNE 0/5 : ⚠️ BLOQUANT POTENTIEL pour l'encadrement. Toute vigilance
  critique disparaît sous pression. Pattern le plus dangereux pour l'encadrement.
→ NULLE global : ❌ BLOQUANT encadrement. Non bloquant management mais limite les impacts croisés.
